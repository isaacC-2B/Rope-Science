# intelligent Rope inspection System (iRiS) Source Code

This is the source code for "iRiS" my portable kernmantle rope inspection system designed to inspect, monitor, and report the condition of kernmantle climbing and work-at-height safety ropes

I would like to express my gratitude to the creators and contributors of the libraries and code editor notebooks used in this project. 

To view and access the models trained for this project please visit the following link as their files were too large to upload: https://drive.google.com/drive/folders/1B4lgCthi87LfwL7KUtQC-k5qoNKmmGT2?usp=sharing
